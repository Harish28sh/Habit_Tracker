# Habit_Tracker

This website is designed to track a person's habits and stores the data in an Excel sheet.

## Website Framework

A simple habit tracker website framework has been added to this repository. The website lets a person add habits by typing a habit name and clicking the Add Habit button. Habits are recorded instantly and saved in the browser storage.

### Included files
- `index.html` — the website page and user interface.
- `styles.css` — styling for the habit tracker layout.
- `script.js` — behavior for adding habits, marking them complete, and exporting the habit list.

### Usage
1. Open `index.html` in a browser.
2. Enter a habit name and click **Add Habit**.
3. Click the status button in the list to mark a habit completed.
4. Use **Export to CSV** to save the habit log for Excel-compatible import.
